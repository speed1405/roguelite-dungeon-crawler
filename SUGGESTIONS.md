# Suggestions

Thank you for playing Roguelite Dungeon Crawler! We welcome your feedback and suggestions to make the game better.

## How to Submit Suggestions

Please feel free to submit suggestions by:
- Opening an issue on GitHub
- Creating a pull request with your proposed changes
- Contacting the project maintainers

## Suggestion Categories

### Gameplay Features
- New biomes or levels
- Additional enemy types
- Power-ups or items
- Character abilities
- Difficulty adjustments

### Improvements
- Performance optimizations
- Visual enhancements
- UI/UX improvements
- Sound and music
- Controls and accessibility

### Bug Reports
- Gameplay bugs
- Collision detection issues
- Rendering problems
- Performance issues

### General Feedback
- Overall game experience
- Balance suggestions
- Quality of life improvements
